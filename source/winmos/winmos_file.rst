:menuselection:`ファイル` メニュー
============================================

新規
----------------------------------
   起動直後の状態に戻ります。
   
   .. hint::
      :kbd:`Ctrl+N` でも操作できます。

.. _file_open:

開く
----------------------------------
   ファイルから分子構造をメインウインドウに読み込みます。各種ソフトのフォーマットに対応しています。
   
   .. hint::
      :kbd:`Ctrl+O` でも操作できます。
      
最近使ったファイル
----------------------------------
   最近開かれたファイルを開きます。

履歴をクリア
^^^^^^^^^^^^^^^
   最近開かれたファイルの履歴を空にします。
      
.. _file_reload_file:

再度読み込み
----------------------------------
   メインウインドウのタイトルに表示されているファイルを再度読み込みます。
   
   .. hint::
       :kbd:`Shift+Ctrl+R` でも操作できます。

.. _file_append:

追加読み込み
----------------------------------
   メインウインドウに表示されている分子構造に、選択したファイルの分子構造を追加します。
   
上書き保存
----------------------------------
   メインウインドウに表示されている分子構造を上書き保存します。
   
   .. hint::
       :kbd:`Ctrl+S` でも操作できます。
       
名前を付けて保存
----------------------------------
   メインウインドウに表示されている分子構造を別名で保存します。
   
   .. hint::
       :kbd:`Shift+Ctrl+S` でも操作できます。
       
.. _file_import:
       
インポート
----------------------------------
   特定の形式の分子構造と、一部の計算結果ファイルを読み込みます。

SMILES
^^^^^^^^^^^^^^^^
   SMILES形式の文字列から分子構造を生成し、メインウインドウに読み込みます。 :guilabel:`Import SMILES` ウインドウが開いたら、テキストボックスにSMILES形式の文字列を入力し、 :guilabel:`Convert` を押してください。内部的には同時にBolloonによる配座探索も実行されます。

MOPAC、GAMESS、Gaussian、NWChem
^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^^
   MOPAC、GAMESS、Gaussian、NWChemの計算結果ファイルを読み込みます。詳細は半経験QM、QMメニュー以下の機能と同等です。
      
エクスポート
----------------------------------
   選択した形式でファイルを出力します。

SMILES 
^^^^^^^^^^^^^^^^^^^^^^^^^^
   メインウインドウに表示されている分子構造をSMILES形式の文字列で出力します。メインウインドウに複数分子表示されている場合は使用できません。Cygwin上でOpenBabelを使用します。
   
   .. include:: ../cygwinwm_required.rst
   
構造式
^^^^^^^^^^
   メインウインドウに表示されている分子構造の構造式の画像をSVG形式で出力します。メインウインドウに複数分子表示されている場合は使用できません。Cygwin上でOpenBabelを使用します。
   
   .. include:: ../cygwinwm_required.rst
   
画像
^^^^^^^^^^^^
   メインウインドウに表示されている内容をBMPまたはJPG形式で出力します。
VRML
^^^^^^^^^^^^
   メインウインドウに表示されている分子構造をVRML形式で出力します。
CHARMM crd File
^^^^^^^^^^^^^^^^^^
   メインウインドウに表示されている分子構造をCHARMMのcrd形式で座標を出力します。

.. _file_editor:

エディタ起動
----------------------------------
   メインウインドウのタイトルに表示されるファイルを、環境設定ウインドウにて選択したテキストエディタで開きます。
   
   .. note::
      テキストエディタで編集後、:ref:`file_reload_file` を選択すると、変更をメインウインドウに反映することができます。
      
フォルダを開く
----------------------------------
   メインウインドウのタイトルに表示されているファイルがの一階層上のディレクトリを開きます。
   
XYZ形式で保存
----------------------------------
   MOPAC形式で保存する際の座標形式を指定します。チェックが入っている場合はXYZ形式、入っていない場合はZ-Matrix形式で保存されます。 :ref:`座標表示エリア <gui_main>` の表示形式も同様に変化します。
   
   .. hint::
      :ref:`座標入力エリア <gui_main>` の :guilabel:`XYZ` チェックボックスでも操作できます。
      
マイプロファイル
----------------------------------

読み込み
^^^^^^^^^^
   ユーザ設定を :file:`UserPref` フォルダの :file:`winmos_mprof.wmx` と :file:`wmset_myprof.ini` から読み込みます。
保存
^^^^^^^^^
   ユーザ設定を :file:`UserPref` フォルダの :file:`winmos_mprof.wmx` と :file:`wmset_myprof.ini` として保存します。
デフォルトに戻す
^^^^^^^^^^^^^^^^^^^
   :file:`UserPref` フォルダの :file:`atoms1.wmx` と :file:`wmset.ini` に記録されているユーザ設定をデフォルト値に戻します。
      
ショートカット作成
----------------------------------
   :file:`winmostar.exe` のショートカットをデスクトップに作成します。
   
終了
----------------------------------
   Winmostarを終了します。
